# ROC ShiftOps

Shift scheduling and management system for the ROC department (ISOC & SCE).

## Deployed on Cloudflare Pages

This repository is connected to Cloudflare Pages for automatic deployment.
Every push to `main` triggers a new deployment automatically.

## Current version

Phase 1 — Frontend prototype (fully functional UI, scheduling engine, PIN login).
Phase 2 — Backend with Cloudflare Workers + D1 database (persistent, multi-user) coming soon.
